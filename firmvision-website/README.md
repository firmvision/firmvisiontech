# Firmvision Technologies Website

A modern React + Tailwind website for Firmvision Technologies.

## Local setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Cloudflare Pages settings

- Framework preset: Vite
- Build command: `npm run build`
- Build output directory: `dist`

## Vercel / Netlify

Both can detect Vite automatically. Use:

- Build command: `npm run build`
- Publish directory: `dist`
